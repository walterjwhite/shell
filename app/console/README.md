# Console

## Console context is set on:
1. initialization
2. when switching directories
3. after some time interval
4. when manually requesting to reload

## Plugins can be set in:
~/.config/walterjwhite/shell
	i.e. If you want to set the JAVA_HOME for zsh, place a configuration file in ~/.config/walterjwhite/shell/zsh/java
		JAVA_HOME=/usr/lib/openjdk19

		And, the JAVA_HOME environment variable will be set accordingly
